Requisitos e instalación

  1. Compilador: GCC 14+
  2. Dependencias:
    • CMake 3.29+
    • pkg-config 0.29+
    • SDL3
    • SDL3_ttf
    • catch2_3

  3. Instalación:

    git clone https://github.com/CS1103/projecto-final-turinmachin
    cd projecto-final-turinmachin
    cmake -S . -B build
    cd build
    cmake --build .
    ./brain_ager
