\doxysubsection{Detailed Description}
A non-\/owning string class (similar to the forthcoming std\+::string\+\_\+view) Note that, because a \doxylink{classCatch_1_1StringRef}{String\+Ref} may be a substring of another string, it may not be null terminated.
