\begin{DoxyItemize}
\item {\bfseries{Cómo ejecutar}}\+: {\ttfamily ./build/brain\+\_\+ager}
\item {\bfseries{Casos de prueba}}\+:
\begin{DoxyItemize}
\item Test unitario de capa \doxylink{classDense}{Dense}, al verificar que una capa \doxylink{classDense}{Dense} y una capa de activación \doxylink{classSigmoid}{Sigmoid} produce una salida del tamaño correcto y con valores entre 0 y 1, validando la propagación hacia adelante.
\item Test de función de activación Re\+LU, para aproximar funciones lineales \$(f(x) = 2x)\$, no lineales \$(f(x) = x\texorpdfstring{$^\wedge$}{\string^}2)\$ y periódicas \$(f(x) = sin(x))\$, evaluando que la salida predicha se acerca suficientemente al valor esperado.
\item Se entrena una red con múltiples épocas sobre el dataset XOR. Se valida que la función de pérdida disminuye y que los resultados se ajustan a los valores esperados (\$\texorpdfstring{$<$}{<}0.\+2\$ para \$0\$ y \$\texorpdfstring{$>$}{>}0.\+8\$ para \$1\$).
\item Verifica que una red neuronal sin capas devuelve exactamente el mismo tensor que recibe como entrada.
\end{DoxyItemize}
\end{DoxyItemize}
