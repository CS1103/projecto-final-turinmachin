\doxysubsection{Detailed Description}
Capa de activación de Rectified Linear Unit (\doxylink{classutec_1_1neural__network_1_1ReLU}{Re\+LU}). Los valores negativos del input se convierten en 0. Los valores no negativos permanecen igual. 

Capa totalmente conectada (o \doxylink{classutec_1_1neural__network_1_1Dense}{Dense} Layer) de una red neuronal. Cada neurona de esta capa se conecta con todas las salidas de la capa anterior. Esta capa tiene pesos y biases como parámetros entrenables.
